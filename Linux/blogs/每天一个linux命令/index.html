more命令

作用
基于vi编辑文本过滤器，以全屏幕分页显示文件内容，支持vi模式下关键字定位操作。
快捷键功能
enter :显示文本下一行内容
space : 显示下一屏文本
q:退出more命令
b:返回上一屏文本内容
h:显示帮助命令
按斜线符/:接着输入一个模式，可以在文本中寻找下一个相匹配的模式。

选项
-d :输出帮助信息
-c :不进行滚屏，每次刷新这个屏幕。
-s :合并多余的空白行。
-u :禁止下划线
-n ：n代表每一屏展示的文本行数。

示例
